comm
====

This is a library for communication systems.

Dependencies
------------

Required libraries:

- CMake
- Boost

Building
--------

```shell
git clone https://github.com/spoonb/comm.git
cd comm
mkdir build
cd build
cmake -DCMAKE_BUILD_TYPE=Release -DCMAKE_INSTALL_PREFIX=~/.local/ ..
make
make test
make install
```
